# ポートフォリオサイトを作った話

**2026.02.14（金）**

## やったこと

GitHub Pages でポートフォリオサイトを公開しました！

https://TORAKO123.github.io/

## きっかけ

`TORAKO123.github.io` のリポジトリはあったけど、中身はデプロイ用のワークフロー（GitHub Actions）だけで空っぽの状態。せっかくなのでポートフォリオを作ることにしました。

## 制作の流れ

### 1. ポートフォリオサイトの作成

Claude Code（AI）と一緒に、1ファイル（`index.html`）でポートフォリオを構築。

**サイトの構成：**

- **Hero** - 自己紹介・SNSリンク
- **About** - 3枚のカードでTORAKOの特徴を紹介
- **Tools** - ComfyUI などの使用ツール
- **Journey** - 2025.5〜のタイムライン
- **Gallery** - 作品ギャラリー
- **Contact** - お問い合わせ案内

**デザインのポイント：**

- パステルピンク × ラベンダーのかわいいカラー
- Zen Maru Gothic & Quicksand フォント
- スクロール時のふわっとしたフェードインアニメーション
- スマホ対応（レスポンシブ + ハンバーガーメニュー）
- 外部ライブラリなし、HTML + CSS + JS のみ

### 2. ギャラリーに作品を追加

X の投稿埋め込みでギャラリーセクションを実装。

- カード型のグリッドレイアウト
- 「Animation」タグでジャンル表示
- 「もっと見る」で X プロフィールへ誘導

## 使った技術

| 技術 | 用途 |
|------|------|
| HTML / CSS / JS | サイト本体 |
| Google Fonts | Zen Maru Gothic, Quicksand |
| CSS Animations | フェードイン、アバターアニメーション |
| IntersectionObserver | スクロール連動の表示制御 |
| X 埋め込み | ギャラリーの作品表示 |
| GitHub Pages | ホスティング |
| GitHub Actions | 自動デプロイ |
